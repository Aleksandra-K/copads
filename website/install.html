Downloading and Installing COPADS

    Collection of Python Algorithms and Data Structures
COPADS can be obtained and installed using one of the 3 methods:

1. Download and Install Using Python setuptools

COPADS can be installed using Python setuptools, simply by typing "easy_install copads" at the command prompt for initial installation.

 

2. Download and Install Using Python Distribution Utilities

COPADS can be downloaded as a package from our download page, un-compress the package and installed at the command prompt with the following command "python setup.py install".

 

3. Obtaining from Code Repository

Assuming that you have a Subversion client installed, COPADS can be checked-out from our Sourceforge SVN repository using the following URL: "https://copads.svn.sourceforge.net/svnroot/copads/copads". After checking out, COPADS can be installed at the command prompt with the following command "python setup.py install".
