Tested Features of COPADS

    Collection of Python Algorithms and Data Structures
Tested features in Version 0.1 (13 February 2009):
  • [Mathematical and Physical Constants] [file = Constants.py]
    • Euler's gamma constant
    • Golden ratio, 2 - Golden ratio
    • Pi, 2 Pi, 180 Pi, Half of Pi
    • Inverse of 180 Pi and 2 Pi
    • Square root of 2 and 2 Pi
    • Natural logarithm of 10, 2, and Pi
    • Inverse of natural logarithm of 10 and 2
  • [Mathematical Operations] [file = Operations.py]
    • Modulus 2, Boolean
  • [Numerical Recipe Functions] [file = NRPy.py]
    • Bessel function I, J, K, Y
    • Modified Bessel function
    • Incomplete Beta function
    • Binomial coefficient
    • Chebyshev evaluation
    • Incomplete Gamma function
    • Complete Gamma function
    • Factorial
    • Natural logarithm of factorial
  • [Statistical Distribution] [file = StatisticsDistribution.py]
    • Normal Distribution
  • [Statistical Hypothesis Tests] [file = HypothesisTest.py]
    • Z-test for a population mean (variance known)
    • Z-test for two population means (variances known and equal)
    • Z-test for two population means (variances known and unequal)
    • Z-test for a proportion (binomial distribution)
    • Z-test for the equality of two proportions (binomial distribution)
    • Z-test for comparing two counts (Poisson distribution)
    • Z-test of a correlation coefficient
    • Z-test for two correlation coefficients
    • Z-test for correlated proportions
    • Spearman rank correlation test (paired observations)
  • [Distances between Objects] [file = ObjectDistances.py]
    • Jaccard, Nei and Li, Dice, Dice Sorensen, Ochiai, Sokal and Michener, Hamming, Euclidean, Kulczynski, Levenshtein
  • [Graph Algorithms] [file = Graph.py]
    • Create edgeless graph (only vertices)
    • Create graph from edge list
    • Find the shortest path between 2 vertices using Dijkstra algorithm
  • [Matrix Algorithms] [file = Matrix.py]
    • Matrix addition, subtraction, multiplication and transposition
    • Element-wise matrix multiplication and division
    • Determinant calculation
    • Matrix minor
    • Conjugate gradient and bi-conjugate gradient calculation on sparse matrix
  • [Tree Data Structures] [file = TreeNodes.py, Tree.py]
    • Insertion and deletion for Red-Black tree
